Protein pI Predictor

Predict isoelectric points (pI) of proteins using a state-of-the-art model that averages predictions from a bidirectional LSTM network and a globally attentive RoPE-based transformer.

This tool accepts a single protein sequence, a FASTA file, or a CSV file of sequences and computes the predicted isoelectric point (pI).
